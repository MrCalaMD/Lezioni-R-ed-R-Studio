R e lanalisi dei dati

Come anche altri programmi R presenta numerosi strumenti per lanalisi dei dati.
Analizzare dati significa:

  1. Ottenere una o pi informazioni da un osservazione
  2. Ordinare queste informazioni possibilmente in tidy data
  3. Pulire le informazioni cos ottenute
  4. Valutare la qualit dellinformazione
  5. Analizzare le informazioni attraverso la statistica descrittiva ed inferenziale
  6. Analizzare le informazioni graficamente

Il problema della raccolta dati

I primi 2 punti sono di solito compito di chi raccoglie i dati mentre gli altri 4 punti sono prerogative di chi il dato lo analizza e cerca di trarne informazioni utlili, fruibili ed immediate.

Spesso la raccolta dei dati gravata da diverse problematiche in quanto le osservazioni possono essere basate su interviste o questionari che debbono essere compilati manualmente attraverso mezzi analogici (carta e penna).

Se possibile sempre meglio ottenere informazioni direttamente da sistemi informatici i quali di solito sono costruiti per presentare osservazioni ed informazioni in modo grezzo, ma corretto.

Esempio pratico

Immaginiamo di aver strutturato un questionario sulla telemedicina e sulle case di comunit ai medici di base di unazienda sanitaria.
Il questionario stato somministrato manualmente ed il medici di base lo hanno compilato con carta e penna; dopodich le risposte sono state caricate su un foglio di lavoro Excel in formato xlsx.

Per fare prima, ad ogni domanda sono state associate delle risposte numerate e solo il numero stato riportato nel file Excel.

Innanzi tutto carichiamo i package necessari per lavorare su questo Dataframe.

#Carico pacchetti necessari

Importare in R un questionario

Il prossimo passo quello di tradurre un questionario in un linguaggio che R sia in grado di interpretare. Il questionario prevede esclusivamente domande che ricavano informazioni categoriche (et, genere, propensione alla telemedicina, ect)

Per portare un questionario in R ho deciso di creare dei vettori categorici, ordinali o meno, assegnando un titolo ad ogni variabile che deve corrispondere al numero del questionario.

Sede <- as.factor(c("1" = "Urbana", "2" = "Rurale"))

Eta <- factor(c("1" = "25-30", "2" = "30-39", "3" = "40-49",
                   "4" = "50-59", "5"= ">60"), 
              levels = c("25-30", "30-39", "40-49", "50-59", ">60"))

N_assistiti<- factor(c("1" = "<500", "2" = "500-1000", 
                          "3" = "1000-1500", "4" = ">1500"),
                     levels = c("<500", "500-1000", "1000-1500",
                                ">1500"))

Genere <- factor(c("1" = "M", "2" = "F"))

Creare una lista dove accogliere i vettori

Per creare una lista uso la funzione vector() che permette di creare liste vuote.
Ho deciso di creare una lista di lunghezza pari al numero di variabili del dataframe. Per farlo ho specificato che la lunghezza della lista doveva essere pari al numero di colonne del dataframe con la funzione ncol().

#Creo un lista vuota
lista <- vector("list", length = ncol(dati))
#Nomino gli elementi della lista come le colonne del DF
for (i in 1:ncol(dati)){
  names(lista)[i]<- colnames(dati[i])
}

Ho poi nominato ogni elemento della lista come le colonne del dataframe usando un ciclo for.
Ma avrei potuto semplicemente scrive: names(lista)=colnames(dati)

Per richiamare gli elementi di un vettore fattoriale possibile avvalersi della funzione base di R parentesi [].
Grazie a questa funzione se ad ogni variabile dato un nome o titolo possibile far restituire ad R un vettore con le variabili corrispondenti.

Convertire le risposte numeriche

Lultimo passaggio consiste nellusare la sintassi base di R ed un ciclo for per convertire tutte le variabili espresse in numero nelle corrispondenti variabili fattoriali.
Per ogni ciclo prelever un vettore dal dataframe con le variabili espresse in numero ed user quel vettore per ottenere le variabili fattoriali prelevate dai vettori ordinati nella lista che li contiene.
Dopo aver ottenuto le variabili le inserir nel dataframe sostituendole alla colonna di partenza.
Per trasferire il vettore nel dataframe user una variabile che sar eliminata ad ogni ciclo for.

for (i in 1:ncol(dati)){
  #genero un vettore fatto dalle variabili fattoriali
  v <- unlist(dati[names(lista)[i]])
  #elimino i nomi (non  fondamentale)
  names(v)<- NULL
  #sovrascrivo con questi dati il DF di partenza
  dati[i]<-lista[[i]][v]
  #elimino il vettore per liberare spazio
  rm(v)
}
